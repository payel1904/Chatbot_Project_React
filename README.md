# React Chatbot

A simple chatbot application built using React. The chatbot allows users to send messages and receive predefined responses in a clean and interactive chat interface.

## Features

- Simple and responsive chat interface
- User and bot message display
- Automatic scrolling to the latest message
- React Hooks (useState, useEffect, useRef)
- Clean UI with custom CSS styling

## Technologies Used

- React
- JavaScript
- HTML
- CSS

## Installation

1. Clone the repository:

```bash
git clone https://github.com/payel1904/Chatbot_Project_React.git
```

2. Navigate to the project folder:

```bash
cd Chatbot_Project_React
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Usage

- Type a message in the input field.
- Click the **Send** button.
- The chatbot will respond with a predefined reply.
